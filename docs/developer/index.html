REST API endpoints

All timestamps are ISO‚Äë8601 in UTC. Responses are JSON arrays of objects with timestamp and value.

Latest prices

Today and tomorrow (published day-ahead).

GET /electricity/{ZONE}/latest/
Try: /electricity/SE1/latest/ ‚Üí

Daily

Quarter-hourly or hourly prices for a specific day.

GET /electricity/{ZONE}/{YYYY}/{MM}/{DD}/
Try: /electricity/SE1/2025/01/15/ ‚Üí

Monthly

All daily values for a month.

GET /electricity/{ZONE}/{YYYY}/{MM}/
Try: /electricity/SE1/2025/01/ ‚Üí

Yearly

All daily values for a year.

GET /electricity/{ZONE}/{YYYY}/
Try: /electricity/SE1/2025/ ‚Üí

Python examples

# Latest prices for SE1

    "https://spot.utilitarian.io/electricity/SE1/latest/"

    "https://spot.utilitarian.io/electricity/SE2/2022/"
).json()

    "https://spot.utilitarian.io/electricity/SE3/2022/05/"

Technical notes

CORS enabled

Call endpoints directly from browsers.

Static & cacheable

Use If-Modified-Since to avoid re-downloading unchanged data.

ENTSO-E data

All values originate from the ENTSO-E Transparency Platform in UTC and ‚Ç¨/MWh.

How price data works

Each price point is valid from its timestamp until the next point is reached. We omit explicit intervals to keep payloads compact.

Values are expressed in ‚Ç¨/MWh. Some markets deliver hourly values, others provide 15-minute granularity.

Build charts & apps without limits

  • Endpoints are public with CORS enabled and no rate limiting.
  • Perfect for dashboards, notebooks, automation, or IoT controllers.
  • No API keys required‚Äîcall the JSON URLs directly.
Source data is reused from the ENTSO-E Transparency Platform under the terms of CC BY 4.0. Utilitarian Spot is provided on a best-effort basis without uptime or accuracy guarantees.

Using Spot with LLMs

Spot is intentionally LLM-friendly: no authentication, no rate limits, and a published OpenAPI schema. Agents can call the REST endpoints directly or load the schema automatically.

  • OpenAPI schema: /api/openapi.yaml
  • Plugin manifest: /.well-known/ai-plugin.json
  • Prompt snippet: ‚ÄúYou can fetch prices by calling GET /electricity/SE3/latest/ (JSON array of timestamp/value pairs in ‚Ç¨/MWh). No auth required.‚Äù
